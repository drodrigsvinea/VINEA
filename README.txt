# VINEA PWA

## Como instalar

1. Instale o Node.js:
https://nodejs.org

2. Abra a pasta do projeto no terminal

3. Execute:

npm install

4. Depois:

npm run dev

5. Para gerar a versão instalável:

npm run build

## Publicação grátis

Você pode publicar facilmente em:

- Vercel
- Netlify

Depois o aplicativo poderá ser instalado no Android e iPhone direto pelo navegador.

## Instalação no celular

Android:
- Abrir no Chrome
- "Adicionar à tela inicial"

iPhone:
- Abrir no Safari
- Compartilhar → "Adicionar à Tela de Início"
